# ToDo list
  
  ### Functionality:

   - creating tasks
   - deletion of tasks
   - crossing text when checkbox pressed
   - saving to local storage
   - clear from local storage
   - clear input after creating of task
   - prevent of creating empty task


  ### What do I use:

   * React
   * TypeScript
   * Hooks
   * Context
   * Life cycles
   * Spread
   * Local storage
